ARIIS - Analyser and Reviser for Incomplete Information Systems

The purpose of this project is aid the software designer alongside the main phases of modelling and development. ARIIS allows that the designer models the software behaviour with incomplete information and verifies if the system model holds some desired specification. If the model does not hold the property, then ARIIS will aid the designer suggesting and correction the model until one, which holds the specification.

To download the source-code and compile go to terminal and type:

$ git clone https://github.com/ARIIS/ariis
$ cd ariis
$ make
$ ./ariis input
